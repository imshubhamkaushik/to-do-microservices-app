# To-Do Microservices App 📝

A simple microservices-based To-Do app with the following services:

- ✅ User Service
- ✅ Task Service
- ✅ Notification Service
- 🐳 Dockerized
- ☸️ Kubernetes-ready

## Technologies

- Node.js
- MongoDB
- Docker
- Kubernetes (Minikube)
- Express.js

## How to Run Locally

```bash
docker-compose up --build
```
